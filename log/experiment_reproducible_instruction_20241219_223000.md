# 植物大战僵尸主题得分规则更新实验记录

## 实验目标
1. 在帮助菜单中添加"得分规则"选项，显示各植物方块的阳光值
2. 将游戏中的"分数"系统改为"阳光"系统
3. 更新得分规则，为不同植物方块设置不同的阳光值

## 实验步骤

### 1. 添加得分规则菜单选项
在`createHelpMenu()`函数中添加新的菜单项：
```cpp
QAction* scoreRules = m_helpMenu->addAction("得分规则");
```

### 2. 实现得分规则显示功能
添加得分规则的连接函数，显示各植物方块的阳光值：
- 🔥 火爆辣椒 (图片1): 125 阳光
- 🔥 火炬树桩 (图片2): 175 阳光  
- 🔫 机枪豌豆 (图片3): 250 阳光
- 🌱 豌豆射手 (图片4): 100 阳光
- 🥜 坚果 (图片5): 50 阳光
- 🌻 双胞向日葵 (图片6): 150 阳光
- 🌻 向日葵 (图片7): 50 阳光
- ❄️ 寒冰射手 (图片8): 150 阳光

### 3. 创建阳光值计算函数
在头文件中声明`getBlockSunlight(int blockType)`函数：
```cpp
int getBlockSunlight(int blockType);
```

在cpp文件中实现该函数，根据方块类型返回对应的阳光值。

### 4. 更新得分计算系统
修改`processElimination()`函数中的得分计算：
```cpp
// 原来：int points = blockType * 10;
// 修改为：int points = getBlockSunlight(blockType);
```

### 5. 更新界面显示文本
将所有"分数"相关的显示文本改为"阳光"：
- 游戏结束界面：玩家1阳光、玩家2阳光、最终阳光
- 游戏进行中界面：玩家1阳光、玩家2阳光、阳光
- 操作指引：消除方块获得阳光值
- 注释：增加玩家阳光值、重置阳光值

## 修改的文件
- `gamewindow.h` - 添加getBlockSunlight函数声明
- `gamewindow.cpp` - 实现得分规则菜单、阳光值计算函数、更新界面显示

## 新增功能
1. **得分规则菜单**：在帮助菜单中新增"得分规则"选项，点击可查看各植物方块的阳光值
2. **阳光值系统**：将原来的简单乘法得分改为基于植物类型的阳光值系统
3. **植物主题**：所有方块都有对应的植物大战僵尸主题名称和阳光值
